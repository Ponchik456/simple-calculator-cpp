# Простой Калькулятор на C++

Простой калькулятор на C++ с автоматическими тестами.

## Возможности

- Базовые операции: сложение, вычитание, умножение, деление
- Возведение в степень (целочисленный показатель)
- Вычисление факториала
- Проверка числа на простоту
- Обработка ошибок (деление на ноль, отрицательный факториал)
- Автоматические модульные тесты

## Требования

- CMake 3.10+

## Сборка

```bash
cd C:\calc
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

## Запускаем калькулятор (из C:\calc\build)
```bash
.\Release\calculator.exe
```

## Запускаем тесты (из C:\calc\build)
```bash
.\Release\tests.exe
```
