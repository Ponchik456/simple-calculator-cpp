# Simple Calculator on C++

Простой консольный калькулятор на C++ с автоматическими тестами.

## Возможности

- Базовые операции: сложение, вычитание, умножение, деление
- Возведение в степень (целочисленный показатель)
- Вычисление факториала
- Проверка числа на простоту
- Обработка ошибок (деление на ноль, отрицательный факториал)
- Автоматические модульные тесты

## Требования

- Компилятор с поддержкой C++11 (g++, clang++, MSVC)
- CMake 3.10+ (опционально)

## Сборка и запуск


```bash
# Компиляция программы
mkdir build && cd build
cmake ..
make

# Запуск
./calculator

# Запуск тестов
./tests
